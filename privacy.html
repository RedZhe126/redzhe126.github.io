隱私權政策

最後更新日期：2026年6月5日

「食決 ShiJue」（以下簡稱「本應用程式」）由開發者 HungChe Hsu（以下簡稱「我們」）開發。我們非常重視您的隱私，並致力於保護您的個人資料。本隱私權政策旨在說明我們如何收集、使用、披露及保護您的資訊，以符合 Apple App Store 的隱私清單 (Privacy Manifest) 要求及相關法規。

1. 我們收集的資訊

為了提供穩定的服務，我們根據「隱私清單」規範收集以下資訊：

  • 帳戶與識別資訊： 當您註冊或登入時，我們收集您的電子郵件地址、顯示名稱、個人頭像。這些資料與您的帳戶連結，用於 App 功能運作及跨裝置同步收藏。
  • 精確位置資訊： 為了提供附近的餐廳搜尋與導覽服務，本應用程式會要求存取您的精確位置。此資訊僅用於提供即時搜尋服務，不會用於追蹤您的長期動向。
  • 訂閱與購買資訊： 為了管理您的「Foodie Pro」訂閱身分，我們會記錄您的購買狀態、方案類型及到期日。支付過程完全透過 Apple App Store 處理，我們不會接觸或儲存您的信用卡資訊。
  • 裝置與使用數據： 包括搜尋紀錄（用於計算每日限額）、收藏列表、裝置識別碼及當機回報，用於優化產品效能與安全防護。

2. 資料安全與加密技術

我們採取業界標準的加密技術來保護您的資料：

  • 傳輸加密： 所有與伺服器的通訊均透過安全加密的 HTTPS (TLS) 連線。
  • 本地儲存加密： 使用者敏感資料（如顯示名稱、頭像識別碼）在裝置端皆使用硬體級加密技術儲存，確保資料安全。
  • 雲端安全： 雲端資料庫儲存於 Google Firebase 與 Apple CloudKit，具備高度安全性與備份機制。

3. 資訊的使用方式

  • 提供、維持及優化服務。
  • 進行身分驗證，確保帳戶與訂閱權限安全。
  • 提供個人化的內容（如收藏、推薦、自訂地圖）。
  • 提供推播通知，告知您最新的服務動態或功能更新。
  • 用於防止不當使用、詐欺或非法攻擊。

4. 資訊的分享與披露

我們不會將您的個人資料出售給第三方。我們僅在以下情況分享資訊：

  • 服務供應商： 我們使用 Google Firebase 作為後端儲存、Apple CloudKit 作為 iOS 快取與同步服務，以及 Apple Maps 與 Google Maps API 提供圖資服務。
  • 法律要求： 若法律有明文規定或為了配合執法機關調查。

5. 資料保存與刪除

我們僅在提供服務所需的期間內保留您的資料。您可以隨時透過應用程式內的設定或聯絡開發者，要求刪除您的帳戶及所有相關資料。一旦刪除帳戶，所有雲端同步的收藏與設定將永久移除。

6. 第三方 SDK 與服務

本應用程式包含以下第三方服務，其隱私政策請參閱各該網站：

  • Google Firebase (Authentication, Firestore, Analytics, Remote Config)
  • Google Maps API & Google Places API
  • Apple MapKit API & CloudKit
  • Google AdMob (用於顯示廣告)

7. 兒童隱私

本應用程式不針對 13 歲以下的兒童。若您發現您的孩子在未經許可下提供資料，請聯繫我們，我們將立即刪除。

8. 政策變更

我們可能會不時更新本政策。重大變更將透過應用程式內公告通知。
